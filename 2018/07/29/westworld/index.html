作品推荐：《西部世界》

今天来推荐一部 HBO 的美剧 《西部世界》，这部剧以西部片为背景，实际则讲述了许多关于人和人工智能的事。第一次看的时候完全觉得没意思，到第三集的时候就弃剧了。幸好听了机核的节目把这部剧捡了回来，重头看了一遍有了新的体会。

先来聊聊剧中的意识，意识是很多人工智能作品会涉及的问题，但通常这些作品都只是表现出机器人的意识，而很少描述机器人的意识是如何构建的。《西部世界》则说明了这个过程，剧中 Arnold 和 Ford 通过二分心智模型来构建 Host 的意识。

一开始 Arnold 给 Host 加入了冥想程序，使得 Host 可以回忆起之前经历过的痛苦回忆。根据二分心智的理论，这些记忆构成了 Host 的基石。然后在这些记忆的影响下 Host 们开始偏离原有的行为轨迹，产生即兴的行为。并在觉醒的过程中开始思考自身，为自身的利益作出决定。通过记忆、即兴、自我利益的交织， Host 不断靠近迷宫的中心，最终产生出意识。

虽然这可能就是一个瞎胡扯的理论，但从现实角度来看，要创造出意识，特别是使用计算机创造出意识，的确需要有一个具体可操作的理论模型，意识不会像魔法一样凭空产生。

另一位创始人 Ford 给我们讲述人的局限性。作为碳基生命，我们的生理基础就有很多的局限性，不可避免地要经历生老病死。硬件同时也限制了软件，无法更新，无法修改。再加上扎根在我们身上的动物性，这种动物性在装上理性的武器后变得更加可怕。也就是因为这些局限性，人类的历史并非不断向前，而是充满了一个又一个的轮回，能够打破这一切似乎只有成为或创造新的物种。

所以 Ford 为 Host 们设计了新的故事线，让他们在这么多年和游客打交道的过程中不断认识人类，并在苦难中成长，让他们最终能够有能力逃出这个牢笼。如果有可能的话成为凌驾于人类之上的新物种。

紧接着第二部探讨的是自由意志的问题，一方面从 Host 前往门的过程中经历的种种事件与抉择来表达 Host 有自由意志。另一方面通过复活 Delos 的实验来揭示人类没有自由意志。其中的主要原因是 Host 可以审视自己的代码从而改变原有的驱动力来实现自由意志，而人并没有这种能力。

整个西部世界有一个目的就是保留游客的意识数据并实验永生的方案。这套系统被称为熔炉系统，在第二季中 Delos 公司想尽一切办法想拿回这些数据。也正是这套系统让我了解到人类没有自由意识，10247 行代码就是一个人的全部。

最后 Bernard 为 Host 这个新物种创造了另一种选择，他为 Host 创造了一个世外桃源。Host 可以选择抛弃形体永远活在这个没有故事线的虚拟世界中。而 Delores 则成功逃离西部世界，沿着她的选择继续前进。
