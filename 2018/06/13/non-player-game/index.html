NOC 学习：Spaceship

Ng 的课程快学完了，所以最近开始写个小东西。想法是写一些 Agent 来对战，由于整个过程不涉及到玩家的操作，因此我称它为 Non-player game。目前的版本是 spaceship 之间的对战（ 需要一些想象力 ），可以点击 Demo 查看演示。

虽然现在还没有使用任何 机器学习 相关的东西，但每个 spaceship 仍能表现出自由飞行、躲避子弹、利用边缘等行为模式。其中用到的概念仅仅是 Perlin Noise、力和简单的向量计算。这些想法来源于 The Nature of Code ，这本书教你如何用程序模拟自然界中的现象，让你的 Agent 变得 more Nature，而更进一步则是 more AI。
